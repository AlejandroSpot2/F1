Alpha Release Overview

This Alpha Release turns the approved proposal into a screenshot-ready static story page. The visuals compare Australia, China, and Japan across 2025 and 2026 using FastF1 race data, with the emphasis on pace, top speed, running-order volatility, and race disruption.

Everything on this page is static by design so the charts can be captured directly for the report and the Notion proposal update. The data was generated locally from FastF1 and written into a browser-ready file so the page can open without a dev server.

Circuit Comparison Bars

These paired charts compare 2025 and 2026 on three race-wide summaries: the median of each driver's best green-flag lap, the fastest green-flag lap in the race, and the median clean-lap speed trap. They serve as the Alpha Release stand-in for the proposal's pace and speed module.

Lap-by-Lap Position Step Charts

Each small multiple tracks the top five finishers in a single race. The side-by-side layout preserves the proposal's season comparison goal while keeping all six races on one page for screenshots.

Position-Change Proxy

This summary compresses all lap-by-lap running-order movement into one value per race. Higher totals indicate a more volatile running order across the full field.

Incident and Neutralization Timeline

Each row shows lap states across an entire race. Green laps indicate uninterrupted running, while yellow, VSC, safety-car, and red-flag segments mark disruption.

Final Circuit Scorecard

This matrix compares 2026 against 2025 for the core metrics at each circuit. Negative lap-time deltas mean 2026 was faster; positive values indicate slower laps. For the other metrics, positive values indicate an increase in 2026.
